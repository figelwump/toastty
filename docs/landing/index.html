A native terminal home for coding agents.

Toastty is a native macOS terminal multiplexer powered by Ghostty and tuned for agent-heavy development: workspaces, live status, unread badges, right-side context, and fast keyboard navigation.

macOS 14.0+ required

Know which agent needs you next.

Workspaces become an agent dashboard. Toastty surfaces ready, working, unread, and later-marked sessions without making you leave the terminal.

Live sidebar status

Claude, Codex, and Pi sessions can report readiness, working state, errors, and unread output.

Jump to active work

Use ⌘⇧A to rotate through unread, approval, ready, and active sessions.

Follow up later

Mark a session for later without pinning it, reordering it, or losing your current flow.

Keep context beside the terminal.

The right panel gives every workspace tab a place for Scratchpad notes, browser pages, and local documents while the main terminal layout stays focused on running work.

  • Open Markdown and code-like local documents without leaving Toastty.
  • Keep a Scratchpad bound to the agent session that produced it.
  • Switch panel tabs independently from terminal tabs and splits.

Keyboard-first, even when the workflow gets wide.

Launch agents, split panes, switch workspaces, open documents, and jump through session state from the command palette and predictable shortcuts.

Native terminal power, shaped for agents.

Toastty stays close to the terminal while adding just enough app structure around it.

Ghostty rendering

GPU-accelerated terminal rendering powered by Ghostty, with Ghostty config compatibility.

Terminal profiles

Launch setups such as tmux, zmx, ssh, or custom scripts from menus and shortcuts.

Split panes

Create, resize, equalize, and zoom terminal panes without losing keyboard focus.
